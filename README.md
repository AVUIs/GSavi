# GSavi
beta code

GS.avi is a gestural instrument that generates continuous spatial visualizations and music from the input of a performer. The features extracted form a performerâ€™s gesture defines the color, position, form and orientation of a 3-demensional Delaunay mesh. The music, composed using granular synthesis, is generated from features extracted from the mesh. The project was created using Processing and MAX/MSP. OSC is used to communicate between the two.

To use this patch, copy and paste the enclosed compressed text into max/msp.
start the processing sketch. 
to begin drawing the mesh, use the trackpad or mouse within the max patch.
